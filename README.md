## WebForms

This project showcases my learning journey in building and structuring **web forms** using HTML. It focuses on understanding form elements, label associations, input attributes, and best practices for creating user-friendly and adaptable forms.

### Key Concepts Covered

* Use of various HTML input types (`text`, `email`, `number`, `date`, `file`, `color`, `tel`, `url`, `password`)
* Applying `label` and `for` attributes for accessibility and clarity
* Working with `checkbox` and `radio` inputs for user choices
* Using `select` dropdown menus and `textarea` for user input
* Implementing `required`, `min`, `max`, and `placeholder` attributes for validation
* Understanding form submission via `action`, `method`, and `enctype` attributes
* Including `hidden` fields and button types (`submit`, `reset`)

### Purpose

This repository serves as a foundation for mastering form handling in HTML — an essential skill for front-end development and user data collection in web applications.
